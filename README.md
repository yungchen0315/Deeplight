# 《潛燈》DEEPLIGHT

深海放置掛機遊戲（像素風）。企劃由 Claude Fable 5 撰寫，實作為純 HTML5、無建置流程、`file://` 可直接開啟的手機優先網頁遊戲，與本 repo 的《皇者天下》完全獨立、互不依賴。

## 核心概念

你是深海觀測站「潛燈號」的站長，唯一的目標是不斷往下潛。採集深海生物的「螢光」餵養燈陣、購買發光模組、升級壓載加速下潛；深度是一條真的會往下捲動的世界，越深、水色越暗、你的燈暈就照得越大；記錄路過的深淵生物進圖鑑，永久提升產量；下潛夠深後「重返海面」轉生，換取永久加成，再潛得更深、更快。

## 如何開始遊玩

用瀏覽器直接打開 `index.html`（建議直式手機畫面）。存檔在瀏覽器 localStorage（key: `deeplight_save_v1`），關閉分頁後離線的時間會在下次打開時自動結算（8 小時全速、之後 16 小時半速，詳見 `src/systems/offlineSystem.js`）。

## 架構

沿用本 repo 另一個專案（《皇者天下》）驗證過的分層哲學：純 `<script>` 標籤依序載入、無 import/export、`window.App` 命名空間，依 `App.Data / App.Models / App.Systems / App.UI / App.Utils` 分層。

```
src/
├─ utils/    通用工具（數字格式化、時間、DOM）
├─ data/     靜態資料（色盤、數值常數、像素圖資料、海域/模組/研究/重構/生物/成就定義）
├─ models/   SaveGame 存檔形狀與工廠函式
├─ systems/  遊戲邏輯（產出、下潛、生物收集、研究、轉生、離線結算、成就、主迴圈）
└─ ui/       畫面渲染與觸控事件，只呼叫 systems
```

美術完全是程式繪製的像素圖（`src/data/sprites.js` 用色鍵網格＋幾何輔助函式手刻），沒有任何外部圖片素材，`src/ui/pixelRenderer.js` 負責把色鍵網格畫到 canvas 上並整數倍放大、關閉平滑。

所有平衡數值（成本成長率、離線公式、轉生公式等）集中在 `src/data/balance.js`，調參不需要動系統邏輯。

## v1.6 內容範圍

**海域與經濟**：11 段海域（透光帶／暮光帶／午夜帶／深淵帶／海溝幽域／無光帶／深海平原／熱泉海淵／裂谷深淵／先驅遺跡／回音迴廊，0–340000m，第 11 段是即將開放的預告牌）、22 種發光模組（各 5 階升級，皆附風味文字）、40 節線性研究、10 項轉生重構升級、44 種深淵圖鑑生物（含 11 種稀有，依目擊次數升 0～5 星，星級決定圖鑑產量加成）、批量購買（x1/x10/xMax／一鍵買滿）。研究／重構／印記／盟約節點都可以點名稱看詳細卡片（成本＋風味文字）。

**三層轉生**：第一層「重返海面」換壓力核心＋重構升級；達到更高門檻後可締結「深淵協約」，換「深淵印記」點 3 分支 × 4 階的印記樹；再往上是「永夜盟約」，換「夜輝」點 4 分支 × 3 階的盟約樹，跟印記樹疊乘增幅，頂端解鎖自動採買／自動過閘門／自動收集生物／自動領任務等自動化，且可在設定裡個別開關。

**留存與獎勵迴圈**：每日簽到（7 天循環）、每日任務（21 種範本抽 3 個）、50 個深度里程碑（潛航紀錄，領取當下按目前產量結算）、金燈魚事件（5 選 2 強力短效增益）、週末活動＋固定日期季節活動（生物出現速度／稀有機率加成）、聲納脈衝主動技能、深淵訊號殘片懸念線、41 個成就、深海珍珠限時加護、事件紀錄時間軸（轉生／協約／盟約／稀有生物初遇等最近 50 筆）、轉生試算面板、存檔備份提醒、版本更新提示。

**體驗與可及性**：程式合成音效與環境音（含音量獨立滑桿）、畫面震動／粒子特效、少數路過生物有 2 幀擺尾動畫、減少動態效果／高對比模式／文字大小三段調整、情境提示系統、可下載的「潛航護照」分享圖卡、遊戲說明彈窗、可安裝的 PWA（離線可玩）、存檔匯出匯入、localStorage 寫入失敗警告、色盲友善的非純色狀態提示（過閘門就緒／目前選中數量）。

詳細設計沿革與尚未實作的規劃項目（店面上架流程等）見 `ROADMAP.md`。
