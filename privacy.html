隱私權政策

《潛燈》DEEPLIGHT（以下稱「本遊戲」）不會收集、儲存或傳輸任何使用者個人資料。

資料儲存

本遊戲的所有進度（存檔）僅儲存在您裝置的瀏覽器 localStorage 中，不會上傳到任何伺服器。解除安裝本遊戲或清除瀏覽器資料會刪除您的存檔，且無法復原（設定頁提供匯出／匯入存檔功能，供您自行備份）。

網路連線

安裝完成後，本遊戲可完全離線遊玩，不會在背景與任何伺服器通訊。

第三方服務

本遊戲不含廣告 SDK、分析工具或任何第三方追蹤程式碼。

聯絡方式

若對本政策有任何疑問，請透過發行者於應用程式商店頁面提供的聯絡方式與我們聯繫。

Privacy Policy

Deeplight ("the game") does not collect, store, or transmit any personal data.

Data Storage

All game progress (save data) is stored only in your device's browser localStorage and is never uploaded to any server. Uninstalling the game or clearing browser data will permanently delete your save (the in-game settings menu offers export/import so you can back it up yourself).

Network Access

Once installed, the game is fully playable offline and never communicates with any server in the background.

Third Parties

The game contains no ad SDKs, analytics, or third-party tracking code of any kind.

Contact

For questions about this policy, please use the contact details provided on the app's store listing.
